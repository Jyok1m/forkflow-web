# forkflow-web

React + TypeScript + Vite app, with Tailwind, React Router, Apollo Client and GraphQL Codegen.

## Requirements

- Node.js 20+
- [pnpm](https://pnpm.io/installation)

## Setup

```bash
pnpm install
```

## Develop

```bash
pnpm dev              # start the dev server
pnpm codegen:watch    # regenerate GraphQL types on change
```

## Other commands

```bash
pnpm build      # type-check and build for production
pnpm preview    # preview the production build
pnpm lint       # run ESLint
pnpm codegen    # generate GraphQL types once
```
